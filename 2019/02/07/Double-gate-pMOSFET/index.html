Double-gate pMOSFET

Si Double-Gate pMOSFETs

Abrstract

​ 对栅极长度$Lg$ = 14,10和7 $nm$的超薄体硅双栅极pMOSFET进行了全面的量子输运研究。 系统地研究了通道长度，体厚，以及限制和运输晶体取向的影响。本文分析了I-V和C-V特性。此外，CMOS性能基准测试结果，如开关延迟，能耗和产品，是以适当方式处理的寄生电阻和电容的影响计算的。发现在长和短沟道长度下晶体取向对驱动电流的影响是显着的，而在Lg = 7nm时，它对有效电容的影响更明显。可以通过平均弹道空穴速度和源 - 漏隧穿比分别很好地解释驱动电流Lg = 14nmandLg = 7nm。同时，有效电容不仅取决于状态密度，还受限于受限方向上孔洞分布的影响。内在和外在性能评估表明，（001）/ [100]和（110）/ [001]是Lg = 7 nm时的最佳限制/传输晶体取向配置，而（110）/ [110]和（ 110）/ [111]是较长栅极长度的最佳选择。随着Lg的缩小，内部和外部性能在缩放的体厚下都会更好。

I. INTRODUCTION

​ 随着CMOS技术的扩展持续到低于10纳米的节点，极小型化晶体管的性能提升变得非常具有挑战性[1]，[2]。根据2017年版的IRDS [3]，多年来在业界广泛使用的硅FinFET将继续在5纳米节点及更高节点存活。然而，在如此小的尺寸下，量子和寄生效应不再是次要问题[4]，[5]。准确而系统地评估其影响对于即将到来的节点的FinFET技术的成功至关重要[6] - [8]。

​ 对于FinFET，其鳍片高度通常远大于其鳍片宽度[7]，并且两个侧壁栅极主导了沟道的静电[9]，因此可以简化为双栅极（DG）MOSFET。有许多关于n型DG-MOSFET的数值研究已经[10] - [18]，而对于它们的p型对应物，迄今为止报道的相对较少[19] - [24]。这些研究要么考虑某些晶体取向和沟道厚度[19]，[20]，要么关注运输性质的某些方面，如弹道速度[20]和空穴迁移[21]。然而，由于强烈的各向异性和非抛物线价带[25]，表面（限制）和通道（传输）方向与不同的体厚相结合将对详细的能带结构[22]和源 - 出 - 隧道效应产生强烈影响。 （SDT）[23];因此，应该在量子迁移水平上更系统地研究它们[24]。此外，大多数关于pMOSFET的研究主要集中在I-V特性[19]，[23]，[24]，而缺乏对C-V特性的详细分析。 C-V特性对于评估重要的CMOS性能指标（如开关延迟和能耗）至关重要。此外，器件寄生效应如串联源极和漏极电阻[4]以及栅极边缘电容[5]在以前的量子传输研究中经常被忽略，但它们对器件性能的影响是显着的[17]，[18]，因为它们变为与小型化器件尺寸产生的沟道电阻和固有电容相当。

​ 在本文中，采用自洽的泊松非平衡格林函数（NEGF）模拟器来捕获所涉及的所有量子效应[26]，[27]。六波段k·p哈密顿量被合并以精确地模拟复杂的价带结构[25]，[28]。对三个通道长度，三个主体厚度和六组晶体取向进行了广泛的模拟。获得了I-V和C-V特性，这些特性也通过平均孔速度（vave），SDT，状态密度（DOS）和电荷分布来解释。内部（寄生效应除外）和外部（包括寄生参数）CMOS能量延迟性能指标进一步量化和比较，设计指南在处理超大尺寸的CMOS性能权衡时优化器件配置。
本文的结构如下。器件结构，晶体取向考虑因素和性能指标将在第II节中讨论。在第III节中，对内部I-V和C-V特性进行了详细的比较和分析，然后分别评估了内部和外部CMOS性能。主要结论见第四节。

Conclusion

​​ 在本文中，使用精确的量子传输模拟完成了对UTB Si pDGFET的缩放限制的全面研究。根据各种沟道长度，体厚和晶体取向组合计算和分析I-V和C-V特性以及内在和外在能量延迟性能。以下是一些关键结论。

1）较长栅极长度的ION主要由ON状态平均弹道空穴速度决定，而在Lg = 7 nm时，SDT比率的计算清楚地表明了这种SDT效应与ION和SS的相关性，即强SDT结果在退化的SS中，因此ION更小。
2）根据包含在哈密顿量中的包含潜在信息的带结构计算的DOS可以很好地解释Ceff从NEGF模拟得到的结果。
3）Ceff可以作为量子电容和静电电容的串联组合来处理。在Lg = 7nm时，Ceff由量子分量控制，而在较长的栅极长度，其值受量子和静电分量的影响。
4）CMOS基准测量指标，包括开关延迟和能耗，与器件的I-V和C-V特性密切相关。此外，它们通过寄生效应显着增加。
5）在内在和外在的情况下，当Lg缩小到7nm时，Tch也应该减小，这意味着应该加强量子限制。
6）在Lg = 7nm时，晶体取向配置的影响变得占主导地位，特别是在大Tch和包括寄生效应时。为了改善这种短门情况下的器件性能，应首先考虑（001）/ [100]和（110）/ [001]，尽管（110）/ [1̄10]和（110）/ [1̄11]是Lg = 10和14 nm时的最佳取向配置。
